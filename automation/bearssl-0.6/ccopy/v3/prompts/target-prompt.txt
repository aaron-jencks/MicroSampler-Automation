```
void
br_ccopy(uint32_t ctl, void *dst, const void *src, size_t len)
{
	unsigned char *d;
    	const unsigned char *s;
	d = dst;
	s = src;
	while (len -- > 0) {
		uint32_t x, y;

		x = *s ++;
		y = *d;
		*d = MUX(ctl, x, y);
		d ++;
	}
}
```

This code does not make use of the dummy buffer provided by the trial context, so you should ignore it. The function will be called as follows:

```
... (
    trial_context_t ctx
) {
    ...
    br_ccopy(*ctx.key, ctx.dest, ctx.data, *ctx.data_len);
    ...
}
```
